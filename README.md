# Week 1 - Retail Sales Excel Analysis

## Project Title

Retail Sales Data Cleaning and Basic Analysis in Microsoft Excel

## Project Objective

The objective of this project is to analyze a retail sales dataset using Microsoft Excel. The project focuses on understanding the dataset, cleaning and formatting the data, performing data quality checks, applying Excel formulas, sorting and filtering records, creating summary tables and charts, and deriving meaningful business insights from the sales data.

## Tools Used

* Microsoft Excel
* CSV Dataset
* GitHub

## Dataset Columns

The dataset contains the following columns:

1. Transaction ID
2. Date
3. Customer ID
4. Gender
5. Age
6. Product Category
7. Quantity
8. Price per Unit
9. Total Amount

## Work Completed

The following tasks were completed during this project:

* Opened and explored the retail sales dataset.
* Converted the dataset into an Excel Table.
* Formatted dates, currency values, and column headers.
* Performed data quality checks for blank cells, duplicate rows, and invalid values.
* Added **Calculated Total** and **Amount Check** columns.
* Used Excel formulas such as SUM, AVERAGE, MIN, MAX, COUNT, SUMIF, and IF.
* Performed sorting and filtering on different columns.
* Created summary tables for:

  * Total Sales by Product Category
  * Quantity Sold by Product Category
  * Total Sales by Gender
* Created:

  * Bar Chart
  * Column Chart
  * Pie Chart
* Documented observations based on the analysis.

## Key Observations

* Electronics generated the highest total sales.
* Beauty generated the lowest total sales.
* Female customers contributed slightly more total sales than male customers.
* The average customer age is approximately 41 years.
* The highest transaction amount recorded is $2,000.
* The dataset contains three product categories: Beauty, Clothing, and Electronics.
* The data quality check showed no major issues after cleaning.
